# Branding: Mirrou Creative Studio

The design system governing this application is termed **Dark Luxury v2.0**. It's built to communicate exclusivity, high-end quality, and technological sophistication.

## Core Principles
1. **Deep Contrast**: Predominantly dark backgrounds with stark, readable foreground elements.
2. **Smooth Dynamics**: Fluid, intentional micro-interactions that don't overwhelm the user.
3. **Immersive Hero Sections**: Subtle, code-driven 3D backgrounds (Three.js) that offer a premium, modern feel.

## Typography
- **Primary**: Sans-serif, highly legible fonts utilized for both headings and body text.
- **Scale**: A clearly defined typographic scale ensuring clear hierarchy from H1 (Hero) down to subtle captions.

## Color Palette
- **Backgrounds**: Deep blacks and charcoal grays.
- **Accents**: Subtle neon or pure white to highlight calls-to-action and critical data points.

## Motion Guidelines
- Animations should feel physically grounded (easing, spring physics).
- Avoid erratic movement; focus on slow, deliberate reveals (fade-ins, subtle y-axis translations).

All these rules are enforced within the Tailwind configuration and utility classes found in the project root and `src/index.css`.
