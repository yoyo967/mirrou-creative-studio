## 6. NAMING CONVENTION (binding)

**Internal working files:**
`[DATE]_[BRAND]_[PROJECT]_[VERSION]_[INITIALS].ext`
Example: `260506_MIRROU_LUMINOUS_V1_OY.pdf`

**Public press files:**
`mirrou-{asset}-{descriptor}-{year}.{ext}`
Example: `mirrou-studio-mission-deck-2026.pdf`

**Image files:**
`{category}-{project|descriptor}-{nn}-{format}.jpg`
Example: `luminous-aura-hero-trio-marble-01-16x9.jpg`

No `final_v3_REAL2.jpg` style naming. No `final` in public filenames.
